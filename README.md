# Brainwash
Brainwash is the Brainfuck interpreter and visualizer.

# install
Brainwash needs a gnu g++.

```terminal
$ make install
```

# usage
Usage:./Brainwash -(h|n|s) terget_file  
	--help(-h) :show help  
	--no-visualize(-n) :only print result  
	--speed=(-s=) :demo speed[ms]
